# <img src="https://github.com/user-attachments/assets/3b6f880b-7d65-427a-8fdf-dc84aa16ed06" alt="app logo" height="25"> Tadawa


Welcome to **Tadawa**, a mobile application designed to help users manage their medication schedules and medical appointments efficiently. This app provides timely reminders for medication intake and hospital visits, promoting better health management and adherence to treatment plans.

## 💡 Features

- **User Registration & Login**: Secure account creation and authentication using firebase
- **Profile Management**: Users can easily manage their personal profiles
- **Medication Management**: Add, edit, and delete medications
- **Appointment Scheduling**: Add and manage upcoming hospital appointments
- **Notifications**: Receive timely alerts for medication times and upcoming appointments
- **Supply Management**: Alerts for low medication stock and expiration dates
- **Theme Preferences**: Choose between dark and light mode for better user experience
- **Multilingual Support**: Supports both Arabic and English languages

## ⚙️ Technologies Used

- **Frontend**: Dart (Flutter Framework)
- **Backend**: Firebase database
